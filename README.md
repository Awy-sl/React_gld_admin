# React_gld_admin

### 目录结构
```
   |_public  -- 公共资源
   |_src  -- 源代码
      |_assets -- 公共资源
      |_components -- 组件
      |_config -- 配置
      |_data  -- 网络请求获取的数据
      |_pages  -- 路由组件
      |_redux  -- 状态管理
      |_routes -- 路由配置
      |_utils -- 工具
      |_App.jsx -- 跟组件
      |_index.js -- 程序主入口
```
